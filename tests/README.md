# Test Files

This directory contains all test and debugging files for the Buy & Sell Phone project.

## Test Files Description

- `debug.php` - Basic PHP functionality and database connection test
- `server_test.php` - Server configuration and compatibility test
- `simple_test.php` - Simple router and file inclusion test
- `test_api.php` - API endpoint testing for phone data
- `test_clean_urls.php` - URL rewriting and clean URL testing
- `test_comprehensive.php` - Comprehensive system testing
- `test_forgot_password.php` - Password reset functionality testing
- `test_paths.php` - File path and routing testing
- `test_router.php` - Router functionality testing
- `test_server.php` - Server compatibility testing

## Usage

These files are for development and debugging purposes only. They should not be deployed to production servers.

To run any test file, access it directly through your web browser:
- `http://your-domain/tests/debug.php`
- `http://your-domain/tests/server_test.php`
- etc.

## Note

These files may contain sensitive debugging information and should be kept out of production environments.
